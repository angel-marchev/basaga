English

Loopy's task: create a structure of a feedback system (all explanatory texts and names and faculty number must be written in the file itself) at: https://ncase.me/loopy/v1.1/. Upload the cyclic file here: http://bit.ly/predavane

Examples: sociogram of mutual preferences between a group of people, ecological system "predator-prey", connections between economic indicators, etc.

The finished file in Loopy is saved with the command "Save as file" (not "Save as link").
Must have the extension * .loopy.
Open again with "load from file".
It must contain a working network, together with the relevant explanations. There must be at least 5 elements and at least 6 connections in the network. Protruding "dry" twigs do not count (as in semantic networks).
The file name must contain "name" and "faculty number" and the extension * .loopy.
